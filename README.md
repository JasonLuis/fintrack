# FinTrack

FinTrack é uma aplicação web de controle financeiro pessoal e compartilhado, criada para ajudar no gerenciamento de receitas, despesas, metas e organização financeira do dia a dia.

A proposta do projeto é oferecer uma experiência simples, moderna e intuitiva, permitindo que usuários acompanhem suas finanças de forma mais clara e organizada.

## Status do projeto

🚧 Em desenvolvimento

O projeto ainda está em construção e novas funcionalidades serão adicionadas gradualmente.

Algumas funcionalidades planejadas:

* Autenticação de usuários
* Dashboard financeiro
* Controle de receitas e despesas
* Gerenciamento de categorias
* Controle de cartões
* Metas financeiras
* Relatórios e gráficos
* Conta compartilhada
* Configurações de perfil

As funcionalidades podem sofrer alterações durante a evolução do projeto.

## Executando o projeto

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

Entre na pasta:

```bash
cd fintrack
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Após iniciar, a aplicação estará disponível em:

```txt
http://localhost:5173
```
